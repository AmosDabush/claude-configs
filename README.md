# claude-configs

My Claude Code configuration files and custom commands.

## Structure
- `CLAUDE.md` - Global instructions
- `commands/` - Custom slash commands
- `cmd_db/` - Database query tool and config
- `agents/` - Agent configurations (future)

## Setup

After cloning, install the db tool dependencies:
```bash
cd cmd_db && npm install
```

## Usage

Use `/db` command in Claude Code to switch databases and run queries.
