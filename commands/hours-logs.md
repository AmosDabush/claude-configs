# /hours-logs - מערכת מעקב שעות עבודה

מערכת לניהול שעות עבודה, חישוב שכר וייצור סקריפטים.

**מיקום קבצים:** `~/hours_logs_moh/`

בחר פעולה:

1. **show** - הצג שעות חודש (טבלה)
2. **summary** - סיכום שעות + נוספות לפי MOH
3. **salary** - חישוב שכר צפוי (ברוטו/נטו)
4. **script** - ייצור סקריפטים למילוי (MOH/Oracle)

**פרמטרים אופציונליים:**
- חודש: `12/2025` (ברירת מחדל: חודש נוכחי)
- סקריפט: `moh` / `oracle` / שניהם

**נתוני שכר:** תעריף 159.34₪, נוספות 150%, בסיס 29,000₪
**תקן MOH:** א-ד 8:36, ה 7:36
