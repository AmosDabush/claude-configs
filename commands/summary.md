# /summary - סיכום שעות

סכם שעות עבודה מקובץ JSON של החודש.

**שימוש:**
- `/summary` - חודש נוכחי
- `/summary 12/2025` - חודש ספציפי

**מיקום קבצים:** `~/hours_logs_moh/YYYY-MM.json`

**חישוב לפי MOH:**
- א-ד: תקן 8:36, נוספות מעל זה
- ה: תקן 7:36, נוספות מעל זה

**הצג:**
- טבלת ימים עם שעות
- סה"כ ברוטו, נטו, רגילות, נוספות
- ימי עבודה, ימי חופשה
